# 📟 Pokédex com Inteligência Artificial

Este projeto tem como objetivo desenvolver uma **Pokédex inteligente**, capaz de identificar Pokémons a partir de imagens utilizando **Redes Neurais Convolucionais (CNN)**.

A aplicação permite que o usuário envie uma imagem e receba como resposta o **nome e a descrição do Pokémon**, simulando o funcionamento de uma Pokédex.

---

## 🚀 Funcionalidades

- 📸 Upload de imagem de Pokémon  
- 🧠 Identificação automática utilizando IA  
- 📖 Exibição do nome e descrição  
- 🎨 Interface inspirada na Pokédex  
- ⚡ Aplicação web com Flask  

---

## 📸 Dataset (Importante)

⚠️ As imagens utilizadas para treinamento **não estão incluídas no repositório**.

Para que o projeto funcione corretamente, é necessário:

1. Criar a pasta `dataset/`
2. Organizar as imagens em subpastas com o nome de cada Pokémon

Exemplo:
dataset/
├── golbat/
├── golduck/
├── nidoking/
├── ninetales/
├── paras/
├── persian/
├── primeape/
├── psyduck/
